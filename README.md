# Agrotech Dashboard

Dashboard de gestao agricola com tela de login, dark mode, cadastro de voluntarios, registro de colheitas, historico de producao, graficos inteligentes e relatorios automaticos.

## Como abrir

Abra `index.html` no navegador ou sirva a pasta com um servidor local:

```bash
python -m http.server 4173
```

Depois acesse `http://127.0.0.1:4173/`.
